# 🎵 Tocador de Música

Um projeto simples e interativo de **tocador de música** desenvolvido com **HTML, CSS e JavaScript**.  
O objetivo é praticar manipulação do DOM, eventos e estilização responsiva, criando uma aplicação funcional e agradável.

---

## 🚀 Funcionalidades
- ▶️ Reproduzir, pausar e parar músicas
- ⏭️ Avançar e retroceder faixas
- 🔊 Controle de volume
- 📂 Lista de músicas carregada dinamicamente
- 🎨 Interface responsiva e estilizada com CSS

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** → Estrutura da aplicação
- **CSS3** → Estilização e responsividade
- **JavaScript (ES6+)** → Lógica de reprodução e interação
